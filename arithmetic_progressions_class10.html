01

Sequences, Series & Progressions

Before diving into Arithmetic Progressions, it helps to understand a few foundational ideas.

Sequence

An ordered list of numbers following a definite rule. Each number in the list is called a term. A sequence can be finite (ends) or infinite (goes on forever).

Series

When the terms of a sequence are added together, the result is called a series. For example, 1 + 2 + 3 + 4 + … is the series of natural numbers.

Progression

A progression is a special sequence in which the general (nth) term can be written using a neat mathematical formula.

Note: Every progression is a sequence, but not every sequence is a progression. A random list like 2, 7, 1, 9, 4 is a sequence with no formula — it is not a progression.

02

What is an Arithmetic Progression?

An Arithmetic Progression (AP) is a sequence of numbers in which each term after the first is obtained by adding a fixed constant to the term before it. This fixed constant is the same throughout the entire sequence.

In simpler terms: subtract any term from the one that follows it — if you always get the same answer, the sequence is an AP.

Quick Test: Is 4, 7, 10, 13, 16 an AP?
7 − 4 = 3 ✔   10 − 7 = 3 ✔   13 − 10 = 3 ✔   16 − 13 = 3 ✔
Yes! The difference is always 3, so this is an AP.

Fig. 2 — The AP 4, 7, 10, 13, 16 visualised on a number line. Each jump is exactly +3.

03

Key Terms: First Term & Common Difference

First Term (a)

The very first number in the AP. Denoted by a or a₁. For the AP 3, 6, 9, 12, … the first term is a = 3.

Common Difference (d)

The constant value added to each term to get the next one. d = aₙ − aₙ₋₁. It can be positive, negative, or zero.

Finding Common Difference d = a₂ − a₁ = a₃ − a₂ = a₄ − a₃ = …
// subtract any term from the NEXT term — the result is always d

Types of AP based on common difference

  • d > 0 (Positive): The sequence is increasing. Example: 2, 5, 8, 11, … (d = 3)
  • d < 0 (Negative): The sequence is decreasing. Example: 20, 15, 10, 5, … (d = −5)
  • d = 0 (Zero): All terms are equal (constant AP). Example: 7, 7, 7, 7, … (d = 0)

Fig. 3 — Bar chart visualisation of increasing, decreasing, and constant APs

04

General Form of an AP

If the first term is a and the common difference is d, every AP can be written in the following standard form:

General Form of an AP a, a+d, a+2d, a+3d, …, a+(n−1)d
EXAMPLE — Write an AP with a = 5 and d = −2
1First term: a = 5
2Second term: a + d = 5 + (−2) = 3
3Third term: a + 2d = 5 + 2(−2) = 1
4Fourth term: a + 3d = 5 + 3(−2) = −1

AP: 5, 3, 1, −1, −3, … (a decreasing AP)

05

Finite vs. Infinite AP

Finite AP

Has a definite last term. The number of terms is limited. Example: Days of a week as multiples — 7, 14, 21, 28 (stops at 4 terms).

Infinite AP

Continues without end. There is no last term. Example: 1, 4, 7, 10, 13, … (three dots indicate it goes on forever).

Memory tip: If you can count the number of terms, it's finite. If the sequence trails off with "…", it's infinite.
06

nth Term (General Term) Formula

The nth term (also called the general term) lets you find any term in the AP without listing all the terms before it. This is the most frequently tested formula in board exams.

nth Term Formula aₙ = a + (n − 1) × d

where:
aₙ = nth term (the term you want to find)
a = first term
n = position of the term (1, 2, 3, …)
d = common difference

Fig. 4 — How the nth term formula is derived from the pattern of the AP

EXAMPLE — Find the 15th term of the AP: 7, 11, 15, 19, …
1Identify: a = 7, d = 11 − 7 = 4, n = 15
2Apply formula: a₁₅ = 7 + (15 − 1) × 4
3= 7 + 14 × 4 = 7 + 56 = 63

∴ The 15th term is 63.

EXAMPLE — Which term of the AP 3, 8, 13, 18, … is 78?
1Here a = 3, d = 5, aₙ = 78
278 = 3 + (n − 1) × 5
375 = (n − 1) × 5 → n − 1 = 15 → n = 16

∴ 78 is the 16th term of this AP.

Last Term (l): In a finite AP, the last term is also called l. Using the nth term formula, l = a + (n − 1)d.
07

Sum of First n Terms

Instead of adding every term individually, we use a formula to find the sum of the first n terms of an AP instantly. There are two versions — use whichever is more convenient.

Sum of First n Terms — Two Versions Sₙ = (n/2) × [2a + (n−1)d]
// use when you know: first term (a) and common difference (d)

Sₙ = (n/2) × (a + l)
// use when you know: first term (a) and last term (l)

Fig. 5 — Derivation of the sum formula using Gauss's pairing method: write the AP forwards and backwards, pair each column (always gives a + l), then divide by 2.

EXAMPLE — Find the sum of the first 20 terms of AP: 5, 8, 11, 14, …
1a = 5, d = 3, n = 20
2S₂₀ = (20/2) × [2(5) + (20−1)(3)]
3= 10 × [10 + 57] = 10 × 67 = 670

∴ The sum of the first 20 terms is 670.

EXAMPLE — Sum of the first 100 natural numbers
1AP: 1, 2, 3, …, 100 → a = 1, l = 100, n = 100
2S₁₀₀ = (100/2) × (1 + 100) = 50 × 101 = 5050

∴ The sum is 5050. (This is Gauss's famous result!)

08

Relationship Between aₙ and Sₙ

There is a very useful connection between the nth term and the sum of n terms that helps you find any term if you already know the sum formula.

Key Relationship aₙ = Sₙ − Sₙ₋₁
// The nth term equals the sum of n terms MINUS the sum of (n−1) terms

Special case: a₁ = S₁
EXAMPLE — If Sₙ = 3n² + 5n, find the 8th term
1S₈ = 3(8²) + 5(8) = 192 + 40 = 232
2S₇ = 3(7²) + 5(7) = 147 + 35 = 182
3a₈ = S₈ − S₇ = 232 − 182 = 50

∴ The 8th term is 50.

Fig. 6 — The nth term is the "extra" piece added when going from Sₙ₋₁ to Sₙ

09

Properties of AP

  • If a fixed number is added to (or subtracted from) every term of an AP, the resulting sequence is also an AP with the same common difference.
  • If every term of an AP is multiplied (or divided) by a fixed non-zero number, the result is again an AP, but with a scaled common difference.
  • If three numbers a, b, c are in AP, then 2b = a + c (the middle term is the arithmetic mean of the other two).
  • The sum of terms equidistant from both ends of a finite AP is always equal to the sum of the first and last terms.
  • If you pick terms from an AP at equal intervals, those terms also form an AP.
  • For any AP, the nth term from the end is given by: l − (n − 1)d, where l is the last term.

Fig. 7 — If a, b, c are in AP, then b is the arithmetic mean of a and c

Real-Life Applications of AP

Simple Interest

Interest earned each year on a fixed principal forms an AP since it increases by the same amount every year.

Stacking Objects

Logs stacked in rows where each row has one fewer log than the one below form an AP.

Constant Speed Motion

Distance covered in successive equal time intervals by an object moving at constant speed forms an AP.

Savings Plans

Saving a fixed additional amount each month results in monthly savings forming an AP.

Stadium Seating

Seats in rows of an auditorium often increase by a fixed number per row, forming an AP.

Calendars & Schedules

Days of a specific weekday in a month (e.g., all Mondays) are in AP with d = 7.

Board Exam Tip: Always identify a, d, and n before applying any formula. Write each value clearly in your answer — examiners award marks for correct identification even if calculation has a minor error.
