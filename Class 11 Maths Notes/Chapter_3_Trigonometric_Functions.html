Chapter 3: Trigonometric Functions

The ultimate guide maximizing your learning: covering angular measurement, periodic functions, identities, competitive-level series, and triangle properties.

Part 1

Angles & Their Measurement

Trigonometry starts with the measurement of rotation. An angle is generated by rotating a given ray about its vertex from an initial side to a terminal side.

Sense of Rotation
°

Degree Measure

If a rotation from the initial side to terminal side is (1/360)th of a revolution, the angle has a measure of 1 degree (1°).

• 1 degree (1°) = 60 minutes (60')
• 1 minute (1') = 60 seconds (60")

rad

Radian Measure

The angle subtended at the center of a circle by an arc whose length is equal to the radius of the circle.

Formula: θ = l / r
(Where l = arc length, r = radius)

Conversion Mastery:

• A complete revolution is 360° or 2π radians. Therefore: π radians = 180°
• Radian Measure = (π / 180) × Degree Measure
• Degree Measure = (180 / π) × Radian Measure
Part 2

Functions, Quadrants & Periodicity

By mapping an angle x to a point P(a,b) on a unit circle (radius = 1), we define cos x = a and sin x = b. This extends trigonometry to all real numbers.

The ASTC Rule
Determines the positive ratios in each quadrant.
Function Domain Range Period
sin(x) ℝ (All Real Numbers) [-1, 1] 2π
cos(x) ℝ [-1, 1] 2π
tan(x) ℝ - {x : x = (2n+1)π/2, n ∈ ℤ} ℝ π
cot(x) ℝ - {x : x = nπ, n ∈ ℤ} ℝ π
sec(x) ℝ - {x : x = (2n+1)π/2, n ∈ ℤ} ℝ - (-1, 1) 2π
csc(x) ℝ - {x : x = nπ, n ∈ ℤ} ℝ - (-1, 1) 2π

🔄 Even & Odd Functions (Negative Angles)

Cosine and Secant are Even functions (they absorb the negative sign). The rest are Odd functions.

  • cos(-x) = cos x and sec(-x) = sec x
  • sin(-x) = -sin x
  • tan(-x) = -tan x
  • cot(-x) = -cot x
  • csc(-x) = -csc x

Part 4

Core Identities & Formulas

1. Sum and Difference of Two Angles:

• sin(A ± B) = sin A cos B ± cos A sin B
• cos(A ± B) = cos A cos B ∓ sin A sin B (Note the sign flip!)
• tan(A ± B) = (tan A ± tan B) / (1 ∓ tan A tan B)
• cot(A ± B) = (cot A cot B ∓ 1) / (cot B ± cot A)
2. Multiple Angle Formulas (2x & 3x):

• sin 2x = 2 sin x cos x = (2 tan x) / (1 + tan² x)
• cos 2x = cos² x - sin² x = 2 cos² x - 1 = 1 - 2 sin² x = (1 - tan² x) / (1 + tan² x)
• tan 2x = (2 tan x) / (1 - tan² x)

• sin 3x = 3 sin x - 4 sin³ x
• cos 3x = 4 cos³ x - 3 cos x
• tan 3x = (3 tan x - tan³ x) / (1 - 3 tan² x)
3. Product to Sum Transformations:

• 2 sin A cos B = sin(A + B) + sin(A - B)
• 2 cos A sin B = sin(A + B) - sin(A - B)
• 2 cos A cos B = cos(A + B) + cos(A - B)
• 2 sin A sin B = cos(A - B) - cos(A + B)
4. Sum to Product Transformations (C-D Formulas):

• sin C + sin D = 2 sin((C+D)/2) cos((C-D)/2)
• sin C - sin D = 2 cos((C+D)/2) sin((C-D)/2)
• cos C + cos D = 2 cos((C+D)/2) cos((C-D)/2)
• cos C - cos D = -2 sin((C+D)/2) sin((C-D)/2)
Part 5 (Advanced)

Advanced Properties & Series

📈 Maximum and Minimum Values

For any trigonometric expression in the form a sin(x) + b cos(x), the maximum and minimum values are strictly bounded by:

-√(a² + b²)   ≤   a sin(x) + b cos(x)   ≤   +√(a² + b²)

📐 Conditional Identities (If A + B + C = π)

In any triangle ABC (where the sum of angles is 180° or π radians), the following special identities hold true:

  • sin 2A + sin 2B + sin 2C = 4 sin A sin B sin C
  • cos 2A + cos 2B + cos 2C = -1 - 4 cos A cos B cos C
  • tan A + tan B + tan C = tan A · tan B · tan C
Sum of Trigonometric Series (Angles in A.P.):

When angles form an Arithmetic Progression (first term = x, common diff = h, terms = n):

Sine Series:
sin(x) + sin(x+h) + ... + sin(x+(n-1)h) = sin(x + (n-1)h/2) · [sin(nh/2) / sin(h/2)]

Cosine Series:
cos(x) + cos(x+h) + ... + cos(x+(n-1)h) = cos(x + (n-1)h/2) · [sin(nh/2) / sin(h/2)]
Part 6

Equations & Properties of Triangles

  • Principal Solutions: Solutions lying in the interval [0, 2π).
  • General Solutions: Expressions involving integer n (n ∈ ℤ) giving all periodic solutions.

Sine Solutions

• sin θ = 0 ⇒ θ = nπ

• sin θ = sin α ⇒ θ = nπ + (-1)ⁿα

Cosine Solutions

• cos θ = 0 ⇒ θ = (2n + 1)π/2

• cos θ = cos α ⇒ θ = 2nπ ± α

Tangent Solutions

• tan θ = 0 ⇒ θ = nπ

• tan θ = tan α ⇒ θ = nπ + α

Properties of Triangles

For any triangle ABC, with sides a, b, c opposite to interior angles A, B, C respectively.

1. The Sine Rule:
a / sin A = b / sin B = c / sin C = 2R (where R is the circumradius)

2. The Cosine Rule:
• a² = b² + c² - 2bc cos A ⇒ cos A = (b² + c² - a²) / 2bc
• b² = c² + a² - 2ca cos B ⇒ cos B = (c² + a² - b²) / 2ca
• c² = a² + b² - 2ab cos C ⇒ cos C = (a² + b² - c²) / 2ab

3. Projection Formula:
• a = b cos C + c cos B
• b = c cos A + a cos C
• c = a cos B + b cos A

4. Napier's Analogy (Law of Tangents):
• tan[(B - C) / 2] = [(b - c) / (b + c)] · cot(A / 2)
