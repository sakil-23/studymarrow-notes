Chapter 1: Sets

The ultimate guide maximizing your learning: covering definitions, subsets, operations, Venn diagrams, and advanced competitive cardinality formulas.

Part 1

Introduction & Representation

In mathematics, a Set is a well-defined collection of distinct objects. By "well-defined," we mean that there is a clear rule to determine whether any given object belongs to the collection or not, without any ambiguity.

{ a, b, c }

Roster (Tabular) Form

All elements are explicitly listed out, separated by commas, and enclosed within curly braces. Order does not matter, and elements are not repeated.

Example: Vowels in English = {a, e, i, o, u}

{ x : P(x) }

Set-Builder Form

Instead of listing elements, this form describes the common property P(x) shared by all members of the set.

Example: V = {x : x is a vowel in the English alphabet}

Part 2

Types of Sets

Sets are categorized based on their cardinality (number of elements) and relationships with other sets.

∅

Empty / Null Set

A set containing absolutely no elements. It is denoted by the Greek letter Φ or empty braces { }.

{ x }

Singleton Set

A set that contains exactly one element.

n(A)

Finite & Infinite

Finite: Contains a countable number of elements.
Infinite: Elements never end and cannot be counted entirely.

A = B

Equal Sets

Two sets A and B are strictly equal if they contain the exact same elements (e.g., A ⊆ B and B ⊆ A).

n(A) = n(B)

Equivalent Sets

Two finite sets are equivalent if they have the same number of elements, even if the elements themselves are different.

Part 3

Subsets, Power Sets & Intervals

A set A is a subset of set B (written as A ⊆ B) if every element in A is also present in B. Remember: every set is a subset of itself, and the empty set (Φ) is a subset of all sets.

🧠 Advanced Formula: Subsets vs. Proper Subsets

If set A contains n elements (i.e., n(A) = n), then:

  • Total number of Subsets: 2ⁿ
  • Total number of Proper Subsets: 2ⁿ - 1 (A proper subset is a subset that is strictly smaller than the original set, so we subtract the set itself).
  • Power Set P(A): The collection of all possible subsets of a set A. Therefore, n(P(A)) = 2ⁿ.
  • Universal Set (U): The main foundational set containing all elements and sets relevant to a particular context or problem.

Real Number Intervals

Open Interval (a, b)

Includes all real numbers between a and b, but strictly excludes the endpoints a and b.

Closed Interval [a, b]

Includes all real numbers between a and b, and includes the endpoints a and b.

Semi-Open [a, b) or (a, b]

Includes one endpoint while excluding the other. E.g., [a, b) includes a but excludes b.

Part 4

Set Operations & Venn Diagrams

Venn diagrams visually represent the relationship between sets using closed curves (usually circles) within a rectangle that represents the Universal Set (U).

Union (A ∪ B)
Elements in A, or in B, or in both.
Intersection (A ∩ B)
Elements strictly common to both A and B.
Difference (A - B)
Elements in A but not in B.
  • Disjoint Sets: Two sets are disjoint if they have no elements in common (i.e., A ∩ B = Φ).
  • Complement of a Set (A'): The set of all elements in the Universal set U that are not in set A. Mathematically: A' = U - A.
  • Symmetric Difference (A Δ B): The set of elements in A or in B, but not in both. It equals (A - B) ∪ (B - A).
Part 5

Laws of Algebra & Cardinality Formulas

De Morgan's Laws

1. (A ∪ B)' = A' ∩ B'
2. (A ∩ B)' = A' ∪ B'

Distributive Laws

1. A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
2. A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)

Commutative & Associative

A ∪ B = B ∪ A
A ∪ (B ∪ C) = (A ∪ B) ∪ C

Practical Formulas (Cardinality)

When solving practical word problems regarding populations, subjects, etc., use these formulas for finite sets:

Standard Unions:

• Two Sets: n(A ∪ B) = n(A) + n(B) - n(A ∩ B)
• If Disjoint (no overlap): n(A ∪ B) = n(A) + n(B)

• Three Sets: n(A ∪ B ∪ C) = n(A) + n(B) + n(C) - n(A ∩ B) - n(B ∩ C) - n(A ∩ C) + n(A ∩ B ∩ C)

🏆 Advanced Formulas for Competitive Exams

For sets A, B, and C, these formulas target specific exact overlaps:

  • Number of elements in exactly one of the sets A or B:
    n(A) + n(B) - 2n(A ∩ B)
  • Number of elements in exactly two of the sets A, B, C:
    n(A ∩ B) + n(B ∩ C) + n(A ∩ C) - 3n(A ∩ B ∩ C)
  • Number of elements in exactly one of the sets A, B, C:
    n(A) + n(B) + n(C) - 2[n(A ∩ B) + n(B ∩ C) + n(A ∩ C)] + 3n(A ∩ B ∩ C)
