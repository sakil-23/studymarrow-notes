Chapter 5: Linear Inequalities

The ultimate guide covering algebraic rules, 1D/2D graphical solutions, absolute value inequalities, and the Wavy Curve method.

Part 1

Introduction to Inequalities

In mathematics, two real numbers or algebraic expressions related by the symbols <, >, ≤, or ≥ form an inequality. Unlike an equation (which represents a balance or exact point), an inequality represents a range or a region of values.

< / >

Strict Inequalities

Inequalities that do not include the possibility of being equal. The boundary point is excluded.

Examples: ax + b < 0, or y > 5

≤ / ≥

Slack Inequalities

Inequalities that include the possibility of equality. The boundary point is included.

Examples: ax + b ≤ 0, or y ≥ 5

The Golden Rules of Algebraic Solutions:

Rule 1: Equal numbers may be added to (or subtracted from) both sides of an inequality without affecting the sign of inequality.
Rule 2: Both sides of an inequality can be multiplied (or divided) by the same positive number without altering the sign.
Rule 3: ⚠️ CRITICAL RULE: If you multiply or divide both sides by a negative number, the inequality sign MUST be reversed (flipped).
Example: If -2x < 10, then dividing by -2 gives x > -5.
Part 2

Graphical Solutions in One Variable

Solutions to linear inequalities in one variable can be represented visually on a number line or mathematically using interval notation.

Number Line Representation
Closed circles include the point. Open circles exclude it.

Open Intervals ( a, b )

Used for strict inequalities (< or >). The boundary numbers are not included in the solution set.
e.g., x > 5 is written as x ∈ (5, ∞)

Closed Intervals [ a, b ]

Used for slack inequalities (≤ or ≥). The boundary numbers are included in the solution set.
e.g., x ≤ 5 is written as x ∈ (-∞, 5]

Part 3

Graphical Solutions in Two Variables

A linear equation like ax + by = c forms a straight line on a Cartesian plane, dividing the plane into two halves called Half-Planes. An inequality represents one of these entire half-planes.

Slack Inequality (≤, ≥)
Represented by a SOLID line. The line itself is part of the solution.
Strict Inequality (<, >)
Represented by a DOTTED/DASHED line. The line is excluded.
  • Step 1: Temporarily replace the inequality sign with an "=" sign to draw the boundary line.
  • Step 2: Use a solid line for ≤ or ≥, and a dotted line for < or >.
  • Step 3 (The Test Point): Pick any point not on the line (usually the origin (0,0)). Plug it into the original inequality.
  • Step 4: If the statement is True, shade the half-plane containing the test point. If False, shade the opposite half-plane.
System of Linear Inequalities:

To solve a system of two or more inequalities in two variables, graph the half-plane for each inequality on the same set of axes. The final solution is the common intersecting region of all the shaded half-planes.
Part 4 (Advanced)

Competitive Topics (JEE Level)

📈 The Wavy Curve Method (Method of Intervals)

A fast, systematic way to solve complex rational inequalities like f(x) / g(x) > 0.

  • Step 1: Factorize polynomials in the numerator and denominator into linear factors. Ensure all x coefficients are positive.
  • Step 2: Find the critical points (roots) by equating each factor to zero, and plot them on a number line in ascending order.
  • Step 3: Start from the rightmost region (beyond the largest root) and assign a (+) sign.
  • Step 4: Move left, crossing roots. If the root comes from a factor with an odd power, change the sign (+ to - or - to +). If the root comes from an even power, keep the same sign.

Modulus (Absolute Value) Inequalities

The modulus |x| represents the distance of x from 0 on the number line. Let a be a positive real number:

  • If |x| < a, then the distance is less than a.
    Solution: -a < x < a   [ x ∈ (-a, a) ]
  • If |x| > a, then the distance is greater than a.
    Solution: x < -a OR x > a   [ x ∈ (-∞, -a) ∪ (a, ∞) ]
  • If |x - c| < a, the distance of x from c is less than a.
    Solution: c - a < x < c + a
The Triangle Inequality for Modulus:

• |x + y| ≤ |x| + |y|
• |x - y| ≥ | |x| - |y| |
