Part 1

Random Experiments & Sample Space

In mathematics, probability measures the degree of certainty of the occurrence of an event. It begins with defining the boundaries of what is possible.

🎲

Random Experiment

An experiment whose exact outcome cannot be predicted with certainty in advance, but it has more than one possible outcome (e.g., rolling a die, tossing a coin).

Sample Space (S)

The set of all possible outcomes of a random experiment. Each individual outcome within this set is called a sample point.

Part 2

Events & Their Types

An Event (E) is simply a subset of the sample space S. Any collection of outcomes is an event.

  • Impossible Event: The empty set Φ. The probability is 0.
  • Sure Event: The entire sample space S. The probability is 1.
  • Simple Event: An event containing exactly one sample point.
  • Compound Event: An event containing more than one sample point.
Mutually Exclusive Events
Cannot occur simultaneously (A ∩ B = Φ)
Exhaustive Events
Their union covers the entire sample space (A ∪ B = S)
Algebra of Events:

• Complementary Event (Not A): A' = S - A
• Event 'A or B': A ∪ B (Union)
• Event 'A and B': A ∩ B (Intersection)
• Event 'A but not B': A - B = A ∩ B'
Part 3

Axiomatic Approach & Theorems

The axiomatic approach treats probability mathematically as a function P mapping subsets of S to real numbers, satisfying three basic axioms.

The Three Axioms of Probability

1. For any event E, 0 ≤ P(E) ≤ 1.
2. Probability of the certain event is 1: P(S) = 1.
3. If A and B are mutually exclusive events, then P(A ∪ B) = P(A) + P(B).

The Addition Theorem
We subtract P(A ∩ B) because the intersection is counted twice when adding P(A) and P(B).
Key Formulas:

• Complement: P(A') = 1 - P(A)
• Difference: P(A - B) = P(A) - P(A ∩ B)
• Addition for 3 Events:
  P(A ∪ B ∪ C) = P(A) + P(B) + P(C) - P(A ∩ B) - P(B ∩ C) - P(C ∩ A) + P(A ∩ B ∩ C)
Part 4 (Advanced)

Competitive Prep (JEE Level)

🎲 Odds In Favor and Odds Against

Instead of expressing probability as a fraction of the total, "Odds" express the ratio of favorable outcomes to unfavorable outcomes.

  • Odds in Favor of A:   Ratio = P(A) / P(A')
  • Odds Against A:   Ratio = P(A') / P(A)
  • Conversion: If odds in favor of A are m : n, then P(A) = m / (m + n).

🔄 De Morgan's Laws in Probability

Applying set theory directly to probability distributions:

  • Neither A nor B:
    P(A' ∩ B') = P((A ∪ B)') = 1 - P(A ∪ B)
  • Not A or Not B:
    P(A' ∪ B') = P((A ∩ B)') = 1 - P(A ∩ B)
Crucial Probability Inequalities:

For any two events A and B:
• Max(0, P(A) + P(B) - 1) ≤ P(A ∩ B) ≤ Min(P(A), P(B))
• Max(P(A), P(B)) ≤ P(A ∪ B) ≤ Min(1, P(A) + P(B))
