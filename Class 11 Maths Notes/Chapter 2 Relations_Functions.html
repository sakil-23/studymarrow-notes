Chapter 2: Relations & Functions

The ultimate guide maximizing your learning: covering Cartesian products, visual mappings, standard functions, domain/range calculation rules, and competitive combinatorics.

Part 1

Cartesian Product & Ordered Pairs

Before defining a relation, we need to understand how elements from two sets can be paired together in a specific, fixed sequence.

(a, b)

Ordered Pair

An ordered pair consists of two elements placed in a specific, fixed order. Equality: Two ordered pairs (a, b) and (c, d) are perfectly equal if and only if a = c and b = d.

A × B

Cartesian Product

For two non-empty sets A and B, the cartesian product A × B is the set of all ordered pairs (a, b) where a ∈ A and b ∈ B.
A × B = {(a, b) : a ∈ A, b ∈ B}

Key Properties & Geometric Interpretation:

• If n(A) = p and n(B) = q, then n(A × B) = p × q.
• If A = Φ or B = Φ, then A × B = Φ.
• 2D Space: ℝ × ℝ represents the coordinates of all points in the 2D Cartesian plane.
• 3D Space: ℝ × ℝ × ℝ represents the coordinates of all points in 3-dimensional space.
Part 2

Relations

A Relation (R) from a non-empty set A to a non-empty set B is simply a subset of the cartesian product A × B. This subset is created by defining a specific mathematical relationship between the first element and second element of the ordered pairs.

Visualizing Relations: Arrow Diagrams

An arrow diagram provides a visual mapping of how elements in the Domain point to elements in the Codomain.

  • Domain: The set of all first elements of the ordered pairs in a relation R.
  • Range: The set of all second elements (often called images) of the ordered pairs in relation R.
  • Codomain: The entirety of set B. Note that Range ⊆ Codomain.
  • Inverse Relation (R-1): If R connects A to B, R-1 connects B to A. R-1 = {(b, a) : (a, b) ∈ R}. Therefore, Domain of R = Range of R-1.

🧠 Advanced Formula: Total Relations Possible

Because a relation is simply a subset of the cartesian product A × B, the total number of distinct relations from Set A to Set B equals the number of subsets of A × B.

If n(A) = p and n(B) = q, then n(A × B) = pq.
Therefore, the Total Number of Relations = 2pq

Part 3

Functions

A function is a special, strict type of relation. A relation f from set A to set B is called a function if every element in set A has one and only one image in set B.

Real-Valued Function

A function f: A → B is a "real-valued function" if its codomain B is a subset of real numbers (ℝ).

Real Function

If both the domain A and the codomain B are subsets of real numbers (ℝ), the function is called a strictly "real function."

🏆 Golden Rules for Domain & Range (Calculus Prep)

When finding the domain of a real function f(x) algebraically, remember two critical constraints:

  • Rule 1 (Fractions): The denominator cannot be zero. If f(x) = p(x)/q(x), then q(x) ≠ 0.
  • Rule 2 (Roots): The expression under an even root (like a square root or 4th root) must be non-negative. If f(x) = √g(x), then g(x) ≥ 0.

🧠 Advanced Formula: Total Functions Possible

If set A has p elements and set B has q elements, the total number of distinct functions that can be mapped from A to B is qp.

Part 4

Standard Functions & Their Graphs

Visualizing the behavior of real functions using graphical plots on the Cartesian plane.

Polynomial Function
f(x) = x² (Parabola)

Domain: ℝ
Range: [0, ∞)

Modulus Function
f(x) = |x|

Domain: ℝ
Range: [0, ∞)

Signum Function
f(x) = |x|/x

Domain: ℝ
Range: {-1, 0, 1}

Greatest Integer Function
f(x) = [x] (Step Function)

Domain: ℝ
Range: ℤ (Integers)

Fractional Part Function
f(x) = {x} = x - [x]

Domain: ℝ
Range: [0, 1)

  • Identity Function: f(x) = x. Domain = ℝ, Range = ℝ. Passes straight through the origin at a 45-degree angle.
  • Constant Function: f(x) = c (where c is a constant). Domain = ℝ, Range = {c}. Forms a horizontal line.
  • Rational Function: f(x) = p(x)/q(x). Domain = ℝ minus points where q(x) = 0.
  • Exponential Function (Advanced): f(x) = ax. Domain = ℝ, Range = (0, ∞).
  • Logarithmic Function (Advanced): f(x) = loga(x). Domain = (0, ∞), Range = ℝ.
Part 5

Algebra of Real Functions

You can perform standard mathematical operations on two real functions f(x) and g(x), assuming they share a common domain.

Operations:

• Addition: (f + g)(x) = f(x) + g(x)
• Subtraction: (f - g)(x) = f(x) - g(x)
• Scalar Multiplication: (kf)(x) = k · f(x), where k is a real scalar.
• Multiplication: (f · g)(x) = f(x) · g(x)
• Quotient: (f / g)(x) = f(x) / g(x), provided g(x) ≠ 0
