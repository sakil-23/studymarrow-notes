Chapter 12: Limits & Derivatives

The foundation of Calculus: Left/Right Hand Limits, Evaluation Methods, L'Hôpital's Rule, and the First Principle of Differentiation.

Part 1

The Concept of a Limit

A limit asks a very specific question: "What value does the function f(x) approach as 'x' gets infinitely close to a certain value 'a'?" It does not matter what the function actually equals exactly at 'a'; it only matters what is happening near 'a'.

Approaching a Limit
The limit is 'L' if both left and right approaches converge.
L.H.L.

Left Hand Limit

The value f(x) approaches as x approaches 'a' from values strictly less than 'a'.

limx→a⁻ f(x)

R.H.L.

Right Hand Limit

The value f(x) approaches as x approaches 'a' from values strictly greater than 'a'.

limx→a⁺ f(x)

Existence of Limit:

A limit limx→a f(x) exists if and only if both the LHL and RHL exist and are strictly equal to each other.

limx→a⁻ f(x) = limx→a⁺ f(x) = L
Part 2

Algebra of Limits & Evaluation

If lim f(x) and lim g(x) exist as x approaches 'a', then limits can be distributed over standard arithmetic operations:

  • Sum/Difference: lim [f(x) ± g(x)] = lim f(x) ± lim g(x)
  • Product: lim [f(x) · g(x)] = lim f(x) · lim g(x)
  • Quotient: lim [f(x) / g(x)] = lim f(x) / lim g(x) (Provided lim g(x) ≠ 0)
  • Scalar Multiple: lim [k · f(x)] = k · lim f(x)

How to Evaluate a Limit

1. Direct Substitution

Plug the value x = a directly into the function. If it yields a real, defined number, that is your limit.

2. Factorization (0/0 Form)

If direct substitution gives an indeterminate 0/0 form, factorize the numerator and denominator to cancel out the common (x - a) term causing the zero.

3. Rationalization

If the function contains square roots and yields 0/0, multiply the numerator and denominator by the conjugate of the expression containing the root.

4. Standard Formula Use

Manipulate the expression to match one of the standard theorem formats (like the algebraic or trigonometric limit formulas listed below).

Part 3

Standard Limit Formulas

Algebraic Limit Theorem:

limx→a [ (xⁿ - aⁿ) / (x - a) ] = n · an-1
(Valid for any rational number n)
Trigonometric Limits (x must be in radians):

• limx→0 [ sin(x) / x ] = 1
• limx→0 [ tan(x) / x ] = 1
• limx→0 cos(x) = 1
• limx→0 [ (1 - cos x) / x ] = 0
The Sandwich (Squeeze) Theorem
If g(x) ≤ f(x) ≤ h(x), and lim g(x) = lim h(x) = L, then lim f(x) = L.
Part 4 (Advanced)

Competitive & JEE Preparatory Limits

🚀 L'Hôpital's Rule

If applying direct substitution to limx→a [f(x) / g(x)] results in the indeterminate forms 0/0 or ∞/∞, you can take the derivative of the numerator and the derivative of the denominator separately:

limx→a [f(x) / g(x)] = limx→a [f'(x) / g'(x)]

If it still yields 0/0, you can apply the rule again and take the second derivatives.

Exponential & Logarithmic Limits:

• limx→0 [ (ex - 1) / x ] = 1
• limx→0 [ (ax - 1) / x ] = loge(a)   (or ln a)
• limx→0 [ loge(1 + x) / x ] = 1
• limx→∞ [ (1 + 1/x)x ] = e
• limx→0 [ (1 + x)1/x ] = e
Part 5

Derivatives & The First Principle

A derivative measures the instantaneous rate of change of a quantity. Geometrically, the derivative of a function at a point is the slope of the tangent line to the curve at that exact point.

First Principle Geometry
The secant line PQ turns into the tangent line at P as h approaches 0.
Derivative from First Principle:

The derivative of function f(x) with respect to x, denoted by f'(x) or (d/dx)[f(x)], is:

f'(x) = limh→0 [ ( f(x + h) - f(x) ) / h ]

Algebra of Derivatives

  • Sum/Difference Rule: (d/dx) [u ± v] = du/dx ± dv/dx
  • Product Rule (Leibniz Rule): (d/dx) [u · v] = u·(dv/dx) + v·(du/dx)
  • Quotient Rule: (d/dx) [u / v] = [ v·(du/dx) - u·(dv/dx) ] / v²
