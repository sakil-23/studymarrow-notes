Chapter 6: Permutations & Combinations

The ultimate guide mastering the art of counting: Arrangements, Selections, Derangements, and Geometric Combinatorics.

Part 1

Fundamental Principles of Counting

Before jumping into formulas, combinatorics relies on two foundational logic rules that govern how we count independent or dependent events.

AND (×)

Multiplication Principle

If an event can occur in m different ways, and following which another event can occur in n different ways, then the total number of occurrences of the events in the given order is m × n.

OR (+)

Addition Principle

If an event can occur in m ways, and another mutually exclusive event can occur in n ways, then exactly one of the events can occur in m + n ways.

Tree Diagram of Multiplication Principle
2 branches × 3 branches = 6 total possible outcomes.

Factorial Notation (n!)

The continued product of the first n natural numbers is called "n factorial".

n! = n × (n - 1) × (n - 2) × ... × 3 × 2 × 1

• Important Property: n! = n × (n - 1)!
• Special Case: 0! = 1
• Factorials of negative numbers or fractions are not defined in standard combinatorics.
Part 2

Permutations (Arrangements)

A Permutation is an arrangement of a number of objects in a definite order. If the order changes, the permutation changes. (e.g., AB is different from BA).

Formula: nPr

The number of permutations of n distinct objects taken r at a time (without repetition):

nPr = n! / (n - r)!

With Repetition Allowed

The number of permutations of n distinct objects taken r at a time, when any object can be repeated any number of times:

Total = nr

Identical Objects

If there are n total objects, where p are of one kind, q of another, and r of a third kind, the arrangements are:

n! / (p! × q! × r!)

Circular Permutations

When objects are arranged in a closed loop (like a round table), there is no fixed "start" or "end". We fix one object to establish a reference point.

Circular Arrangement
Fixing one element ('A') breaks the loop.
1. Distinguishable Direction (e.g., people at a table):
If clockwise and anti-clockwise arrangements are different: (n - 1)!

2. Indistinguishable Direction (e.g., beads on a necklace):
If the loop can be flipped over, making clockwise/anti-clockwise identical: (n - 1)! / 2
Part 3

Combinations (Selections)

A Combination is a selection of objects where order does not matter. (e.g., Selecting team members A and B is the exact same as selecting B and A).

Formula: nCr

The number of combinations of n distinct objects taken r at a time:
nCr = n! / [r! × (n - r)!]

Relationship to Permutation: nPr = nCr × r!

Crucial Properties of nCr

  • Symmetry: nCr = nCn-r (Selecting r objects is the same as leaving behind n-r objects).
  • Equality Rule: If nCx = nCy, then either x = y, or x + y = n.
  • Pascal's Identity: nCr + nCr-1 = n+1Cr
  • Total Selections: The sum of all possible selections (choosing 0, 1, 2... up to n objects) = nC0 + nC1 + ... + nCn = 2ⁿ.
  • Selecting at least one: 2ⁿ - 1.
Part 4 (Advanced)

Grouping & Distribution (JEE Level)

📦 Division into Groups (Distinct Objects)

Dividing (m + n + p) distinct items into three unequal groups of size m, n, and p:

Number of ways = (m + n + p)! / (m! × n! × p!)


If the groups are of equal size (e.g., 3 groups of size m), we must divide by 3! to avoid overcounting identical groups:

Number of ways = (3m)! / [ (m!)³ × 3! ]

⭐ Stars and Bars Method (Identical Objects)

To find the number of ways to distribute n identical objects into r distinct boxes (where a box can receive 0 objects):

Total ways = n+r-1Cr-1

If every box must receive at least 1 object, the formula changes to: n-1Cr-1

Part 5 (Advanced)

Number Theory Applications & Derangements

Number of Divisors

If a natural number N is prime factorized as: N = pa · qb · rc ... (where p, q, r are distinct primes):

• Total number of divisors: (a + 1)(b + 1)(c + 1)...
• Sum of all divisors: [ (pa+1 - 1)/(p - 1) ] × [ (qb+1 - 1)/(q - 1) ] × ...
• Note: Proper divisors = Total Divisors - 2 (excluding 1 and N).

Derangements (Dn)

A derangement is a permutation where none of the objects appear in their "original" or "right" place. (e.g., putting 4 letters into 4 wrong envelopes).

Dn = n! [ 1 - 1/1! + 1/2! - 1/3! + ... + (-1)ⁿ/n! ]

Part 6 (Advanced)

Geometrical Combinatorics

Diagonals in an n-gon
Points = n, Lines = nC2
  • Straight Lines: Given n points (no 3 collinear), the number of lines formed by joining them is nC2.
  • Triangles: Given n points (no 3 collinear), the number of triangles formed is nC3.
  • Collinear Exception: If m out of n points are collinear, you must subtract the lines/triangles they would have formed, but add back 1 for the straight line they all share:
    Lines = nC2 - mC2 + 1
    Triangles = nC3 - mC3
  • Number of Diagonals: For an n-sided polygon, a diagonal connects any two non-adjacent vertices: nC2 - n.
  • Rectangles in a Grid: A grid made of m horizontal lines and n vertical lines contains mC2 × nC2 total rectangles (including squares).
