The ultimate guide covering the Imaginary Unit, Argand Planes, Polar Forms, Unity Roots, and advanced competitive algebra.

Part 1

The Imaginary Unit (iota)

The real number system falls short when trying to solve equations like x² + 1 = 0. To solve this, mathematicians introduced the imaginary unit, denoted by i (iota), where i = √(-1). Therefore, i² = -1.

Cyclic Powers of i:

• i¹ = i
• i² = -1
• i³ = i² · i = -i
• i⁴ = (i²)² = (-1)² = 1

General Rule for integer n:
i4n = 1  |  i4n+1 = i  |  i4n+2 = -1  |  i4n+3 = -i

💡 Note: The sum of four consecutive powers of i is always zero: in + in+1 + in+2 + in+3 = 0.
Part 2

Complex Numbers & Their Algebra

A number of the form z = a + ib, where a and b are real numbers, is a complex number.

  • Real Part: Re(z) = a
  • Imaginary Part: Im(z) = b
  • Purely Real: If b = 0, the number is purely real (z = a).
  • Purely Imaginary: If a = 0, the number is purely imaginary (z = ib).
  • Equality: Two complex numbers z₁ = a + ib and z₂ = c + id are equal if and only if a = c and b = d.
+ / -

Addition & Subtraction

Add/Subtract the real parts together and the imaginary parts together.

(a + ib) ± (c + id) = (a ± c) + i(b ± d)

×

Multiplication

Multiply like polynomials, remembering that i² = -1.

(a + ib)(c + id) = (ac - bd) + i(ad + bc)

Part 3

Modulus & Conjugate

Definitions

For a complex number z = a + ib:

  • Conjugate (z̄): Found by changing the sign of the imaginary part. z̄ = a - ib
  • Modulus (|z|): The absolute distance from the origin. |z| = √(a² + b²)
  • Multiplicative Inverse (z⁻¹): z⁻¹ = 1/z = z̄ / |z|²

Crucial Properties

Conjugate Properties

• (z̄)̄ = z
• z + z̄ = 2 Re(z)
• z - z̄ = 2i Im(z)
• (z₁ ± z₂)̄ = z̄₁ ± z̄₂
• (z₁z₂)̄ = z̄₁ z̄₂
• (z₁/z₂)̄ = z̄₁ / z̄₂

Modulus Properties

• z z̄ = |z|² (Extremely Important)
• |z| = |z̄| = |-z|
• |z₁ z₂| = |z₁| |z₂|
• |z₁ / z₂| = |z₁| / |z₂|
• Triangle Inequality:
  |z₁ + z₂| ≤ |z₁| + |z₂|
  ||z₁| - |z₂|| ≤ |z₁ - z₂|

Part 4

The Argand Plane & Polar Form

A complex number z = x + iy can be represented geometrically as a point P(x, y) on a 2D plane called the Argand Plane (or Complex Plane). The x-axis is the Real Axis, and the y-axis is the Imaginary Axis.

Where α = tan⁻¹(|y/x|)
  • Argument/Amplitude (θ): The angle made by the vector OP with the positive direction of the real axis.
  • Principal Argument: The unique value of θ such that -π < θ ≤ π.
  • Polar Form: Any complex number can be written as z = r(cos θ + i sin θ), where r = |z| and θ is the principal argument.
  • Euler's Form (Advanced): z can also be written using exponentials as z = r·eiθ. Note that eiθ = cos θ + i sin θ.
Part 5

Quadratic Equations

With the advent of complex numbers, we can now find roots for quadratic equations even when the discriminant is negative.

For the standard equation ax² + bx + c = 0, the Discriminant is D = b² - 4ac.

If D < 0, the roots are complex and occur in conjugate pairs:

x = [-b ± i√( |D| )] / 2a
or equivalently,
x = [-b ± i√( 4ac - b² )] / 2a
Part 6 (Advanced)

Advanced Competitive Topics (JEE)

🔄 Cube Roots of Unity

The equation z³ = 1 has three roots: one real (1) and two complex conjugate roots denoted by ω (omega) and ω².

  • ω = (-1 + i√3) / 2
  • ω² = (-1 - i√3) / 2
  • Property 1: 1 + ω + ω² = 0
  • Property 2: ω³ = 1 (and thus ω3n = 1)

📐 De Moivre's Theorem

An extremely powerful theorem for calculating powers and roots of complex numbers in polar form.

(cos θ + i sin θ)ⁿ = cos(nθ) + i sin(nθ)
or in Euler form: (eiθ)ⁿ = einθ

📏 Geometry of Complex Numbers

  • Distance Formula: The distance between two points z₁ and z₂ is given by |z₁ - z₂|.
  • Equation of a Circle: |z - z₀| = r represents a circle with center at complex number z₀ and radius r.
Square Root of a Complex Number (z = a + ib):

√(a + ib) = ± [ √((|z| + a) / 2) + i·sign(b)·√((|z| - a) / 2) ]

Where sign(b) is '+' if b > 0, and '-' if b < 0.
