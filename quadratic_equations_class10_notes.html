Quadratic Equations

Revision Notes — Standard Form • Roots • Methods • Discriminant • Graphs

1. What is a quadratic equation?

A quadratic equation is a polynomial equation of degree 2 in one variable. Any equation that can be expressed in the following standard form is a quadratic equation:

ax² + bx + c = 0
where a, b, c are real numbers and a ≠ 0

Here, a is the coefficient of x², b is the coefficient of x, and c is the constant term. The condition a ≠ 0 is essential — if a were 0, the equation would reduce to a linear equation.

💡 The highest power of the variable in a quadratic equation is always 2. This is why it is also called a “second-degree equation.”

Examples of quadratic equations

• x² − 5x + 6 = 0  (a = 1, b = −5, c = 6)
• 2x² + 3x − 2 = 0  (a = 2, b = 3, c = −2)
• x² − 4 = 0  (a = 1, b = 0, c = −4)

Not a quadratic equation

• x³ + x + 1 = 0  (degree is 3, not 2)
• 2x + 5 = 0  (degree is 1, this is linear)
• 0·x² + 3x + 1 = 0  (a = 0, so not quadratic)

2. Roots of a quadratic equation

A value of x that satisfies a quadratic equation is called a root (or solution) of that equation. If x = α is a root of ax² + bx + c = 0, then substituting α in place of x makes the equation true, i.e., aα² + bα + c = 0.

Every quadratic equation has exactly two roots (which may be equal, distinct, or non-real). The roots of a quadratic equation are the same as the zeroes of the corresponding quadratic polynomial ax² + bx + c.

The two roots are commonly denoted by the Greek letters α (alpha) and β (beta).

3. Methods of solving quadratic equations

There are three standard methods to find the roots of a quadratic equation:

Factorisation
Method 2
Completing the square
Method 3

Method 1 — Factorisation

The quadratic expression is split into two linear factors. Each factor is then set equal to zero to find the two roots.

  1. Write the equation in standard form: ax² + bx + c = 0.
  2. Split the middle term bx into two terms mx + nx such that m + n = b and m × n = ac.
  3. Group and factor to get two linear factors.
  4. Set each factor equal to zero and solve for x.
Example: x² − 5x + 6 = 0 → split −5x as −3x − 2x (since −3 + (−2) = −5, −3 × −2 = 6) → x(x−3) − 2(x−3) = 0 → (x−2)(x−3) = 0 → x = 2 or x = 3

Method 2 — Completing the square

This method converts the equation into a perfect square form, making it easy to find the roots by taking square roots on both sides.

  1. Write the equation as ax² + bx = −c. Divide throughout by a.
  2. Add (b/2a)² to both sides so the left side becomes a perfect square trinomial.
  3. Write the left side as (x + b/2a)².
  4. Take the square root of both sides.
  5. Solve for x by isolating it on one side.

Method 3 — The quadratic formula

This is the most universal method. It works for all quadratic equations regardless of whether the roots are rational, irrational, or non-real. It is derived by applying the completing-the-square method to the general equation ax² + bx + c = 0.

x = (−b ± √(b² − 4ac)) / 2a
The quadratic formula — works for all values of a, b, c

To use this formula, identify the values of a, b, and c from the equation and substitute them in. The ± sign gives two roots — one with + and one with −.

Tip: Use factorisation when the middle term splits easily. Use the quadratic formula when factorisation is not obvious or when roots are likely irrational.

4. The discriminant and nature of roots

The expression D = b² − 4ac inside the square root of the quadratic formula is called the discriminant. It tells us the nature of the roots without actually solving the equation.

D = b² − 4ac
Discriminant — determines what kind of roots the equation has
Value of DNature of rootsDescription
D > 0Two distinct real rootsThe parabola crosses the x-axis at two different points
D = 0Two equal real rootsThe parabola just touches the x-axis at one point (the vertex)
D < 0No real rootsThe parabola does not cross or touch the x-axis at all

When D = 0, both roots are equal: α = β = −b/2a

If D ≥ 0, the roots are real. For real and rational roots (when a, b, c are integers), D must be a perfect square.

5. Graphical representation (parabola)

The graph of a quadratic equation y = ax² + bx + c is always a smooth curved shape called a parabola. The roots of the equation are the x-coordinates of the points where the parabola intersects the x-axis (y = 0).

The direction of opening depends on the sign of a:
• If a > 0 → parabola opens upward (U-shape)
• If a < 0 → parabola opens downward (∩-shape)

D = 0 — Equal roots

D < 0 — No real roots

6. Relationship between roots and coefficients

If α and β are the two roots of the quadratic equation ax² + bx + c = 0, then the following relationships always hold:

Sum of roots
α + β = −b / a
Product of roots
α × β = c / a
Rebuild the equation
x² − (α+β)x + αβ = 0

These relationships are very useful. If the sum and product of roots are known, the quadratic equation can be reconstructed directly without needing to solve for individual root values.

x² − (sum of roots) · x + (product of roots) = 0

7. Forming a quadratic equation from given roots

If we know both roots α and β, the quadratic equation can be formed as:

(x − α)(x − β) = 0
Expanding gives: x² − (α+β)x + αβ = 0
Example: If roots are 3 and −2, then sum = 3 + (−2) = 1 and product = 3 × (−2) = −6. The quadratic equation is x² − x − 6 = 0.

9. Important points to remember

  • A quadratic equation always has exactly two roots (counting multiplicity).
  • The condition a ≠ 0 must always hold; otherwise it is not a quadratic equation.
  • For real roots to exist, the discriminant must satisfy D ≥ 0.
  • If D is a perfect square and a, b, c are integers, the roots are rational numbers.
  • The quadratic formula always works — use it when factorisation is not immediately obvious.
  • Completing the square is the method from which the quadratic formula is derived.
  • In word problems, always check whether both roots are meaningful in context (e.g., reject negative lengths or ages).
  • The graph of a quadratic is always a parabola; the roots are the x-intercepts.
