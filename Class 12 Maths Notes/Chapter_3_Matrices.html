Chapter 3: Matrices

The foundation of linear algebra: Orders, Operations, Symmetry, Invertibility, and Advanced Transformation Techniques.

Part 1

Matrix Definition & Order

A matrix is an ordered rectangular array of numbers or functions. The numbers or functions are called the elements or entries of the matrix. We enclose them in square brackets [ ] or parentheses ( ).

Structure of a Matrix A = [aij]
Where 'i' is the row index and 'j' is the column index.
  • Order of a Matrix: A matrix having m rows and n columns is called a matrix of order m × n (read as "m by n").
  • Total Elements: The total number of elements in an m × n matrix is exactly mn.
  • Equality of Matrices: Two matrices A = [aij] and B = [bij] are equal if they have the same order AND each corresponding element is exactly equal (aij = bij).
Part 2

Classification of Matrices

1. Row & Column Matrices

A matrix is a Row Matrix if it has only one row (1 × n). It is a Column Matrix if it has only one column (m × 1).

2. Square Matrix

A matrix in which the number of rows is equal to the number of columns (m = n). Elements where i = j form the principal diagonal.

3. Diagonal Matrix

A square matrix where all non-diagonal elements are zero. (aij = 0 for i ≠ j).

4. Scalar Matrix

A diagonal matrix where all diagonal elements are equal to the same constant k.

5. Identity Matrix (I)

A scalar matrix where all diagonal elements are exactly 1. It acts like the number '1' in matrix multiplication.

6. Zero Matrix (O)

A matrix (of any order) where every single element is zero.

Part 3

Operations on Matrices

Addition & Scalar Multiplication:

• Addition (A + B): Matrices must have the same order. Add corresponding elements (aij + bij). Matrix addition is commutative (A+B = B+A) and associative.
• Scalar Mult (kA): Multiply every single element inside the matrix by the constant 'k'.

Matrix Multiplication

The multiplication of two matrices A and B is defined ONLY if the number of columns in A equals the number of rows in B.

Multiplication Dimensional Rule
The inner dimensions must match. The outer dimensions form the new order.
  • Row by Column Method: To find element cij of the product matrix, multiply the ith row of matrix A by the jth column of matrix B element-by-element, and sum the products.
  • Non-Commutative: Generally, AB ≠ BA. Matrix multiplication is NOT commutative.
  • Identity Rule: AI = IA = A for any square matrix A.
  • Zero Product: It is possible that AB = O even if neither A nor B is a zero matrix.
Part 4

Transpose & Symmetry

The transpose of a matrix A, denoted by AT or A', is obtained by interchanging its rows and columns.

Properties of Transpose

• (AT)T = A
• (kA)T = k(AT)
• (A + B)T = AT + BT
• Reversal Law: (AB)T = BT AT

Transposing an Order
A (3 × 2) matrix becomes a (2 × 3) matrix.
  • Symmetric Matrix: A square matrix is symmetric if AT = A. (Elements are mirrored across the main diagonal: aij = aji).
  • Skew-Symmetric Matrix: A square matrix is skew-symmetric if AT = -A. (aij = -aji).
    Crucial Note: All main diagonal elements of a skew-symmetric matrix are always zero!
The Decomposition Theorem:

Any square matrix 'A' can be expressed as the sum of a symmetric and a skew-symmetric matrix:

A = ½(A + AT) + ½(A - AT)

(Where the first part is Symmetric, and the second part is Skew-Symmetric).
Part 5

Invertible Matrices

If A is a square matrix, and there exists another square matrix B of the same order such that AB = BA = I, then B is called the inverse of A, denoted by A-1.

  • Uniqueness: Inverse of a square matrix, if it exists, is unique.
  • Reversal Law of Inverse: (AB)-1 = B-1 A-1
  • Inverse can be found using Elementary Row Operations (A = IA) or Elementary Column Operations (A = AI) until the left side becomes the Identity matrix.
Part 6 (Advanced)

Competitive Topics (JEE Level)

🎯 Trace of a Matrix (Tr)

The Trace of a square matrix A is the sum of its principal diagonal elements: Tr(A) = Σ aii.

  • Tr(A + B) = Tr(A) + Tr(B)
  • Tr(kA) = k × Tr(A)
  • Tr(AB) = Tr(BA)    (Even if AB ≠ BA)

📈 Special Square Matrices to Memorize

  • Orthogonal Matrix: A × AT = I (and AT × A = I). For these matrices, A-1 = AT.
  • Idempotent Matrix: A² = A. (Multiplying it by itself changes nothing).
  • Involutory Matrix: A² = I. (The matrix is its own inverse).
  • Nilpotent Matrix: Ak = O for some positive integer k. (The smallest such 'k' is called its index).
