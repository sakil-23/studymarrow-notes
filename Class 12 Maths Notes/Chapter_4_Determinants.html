Chapter 4: Determinants

Extracting numbers from matrices: Properties, Minors, Adjoints, Inverses, and solving complex Systems of Linear Equations.

Part 1

The Determinant Concept

To every square matrix A = [aij] of order n, we can associate a unique real or complex number called the determinant of the square matrix A. It is denoted by |A|, det(A), or Δ.

Note: Only square matrices have determinants. The symbol |A| means determinant, NOT absolute value!

Sign Convention for Expansion
Determined by (-1)i+j for row 'i' and column 'j'.

Order 2 × 2

For matrix A = [a b; c d]:

|A| = (a × d) - (b × c)

Order 3 × 3

Expanding along Row 1 (R₁):

|A| = a₁₁(M₁₁) - a₁₂(M₁₂) + a₁₃(M₁₃)

*You can expand along ANY row or column. It is best to expand along the one with the most zeros!

Part 2

Core Properties of Determinants

These properties allow us to simplify massive determinants without fully expanding them.

  • Property 1 (Reflection): The value of the determinant remains completely unchanged if its rows and columns are interchanged. (|A| = |AT|)
  • Property 2 (Switching): If any two adjacent rows (or columns) of a determinant are interchanged, the sign of the determinant changes. (Value becomes -|A|).
  • Property 3 (Proportionality): If any two rows (or columns) of a determinant are identical or proportional, then the value of the determinant is zero.
  • Property 4 (Scalar Multiplier): If each element of a single row (or column) is multiplied by a constant k, the value of the entire determinant gets multiplied by k.
  • Property 5 (Addition): If elements of a row/column are expressed as sum of two terms, the determinant can be split into two separate determinants.
  • Property 6 (Invariance): If we apply the elementary operation Ri → Ri + kRj, the value of the determinant remains totally unchanged.
Part 3

Minors, Cofactors & Adjoint

Finding a Minor
Delete the row and column of the element to find the remaining sub-determinant.

Minor (Mij)

The Minor of an element aij is the determinant of the sub-matrix obtained by deleting its ith row and jth column.

Cofactor (Cij or Aij)

The Cofactor is simply the Minor multiplied by the sign convention.

Cij = (-1)i+j × Mij

Adjoint of a Matrix (adj A):

The adjoint of a square matrix A is defined as the Transpose of the matrix formed by its Cofactors.

If A = [aij] and C = [Cij], then adj(A) = CT
Part 4

Inverse & System of Linear Equations

A square matrix A is invertible (it has an inverse) if and only if it is a non-singular matrix (meaning |A| ≠ 0).

The Inverse Formula:

A-1 = (1 / |A|) × adj(A)

Crucial Reversal Property: (AB)-1 = B-1 A-1

Part 5 (Advanced)

Competitive Properties (JEE Level)

📈 Advanced Properties of Adjoint & Inverse

For a square matrix A of order n × n:

  • Fundamental Theorem: A × (adj A) = (adj A) × A = |A| × I
  • Determinant of Adjoint: |adj(A)| = |A|n-1
  • Double Adjoint Determinant: |adj(adj(A))| = |A|(n-1)²
  • Matrix Double Adjoint: adj(adj(A)) = |A|n-2 × A

🎯 The Scalar Pull-Out Rule

If you multiply a matrix A of order n by a scalar k, and then take the determinant, the scalar is pulled out n times (once for each row):

|kA| = kn |A|

Determinant of Special Matrices:

• Skew-Symmetric Matrix: If A is skew-symmetric (AT = -A) and its order 'n' is an odd number, then exactly |A| = 0.
• Orthogonal Matrix: If A × AT = I, then |A| = ±1.
• Idempotent Matrix: If A² = A, then |A| = 0 or 1.
