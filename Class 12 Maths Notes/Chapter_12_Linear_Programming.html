Chapter 12: Linear Programming

The ultimate guide to optimization: Constraints, Bounded Regions, Corner Point Methods, and advanced Iso-Profit techniques.

Part 1

Core Terminology of LPP

Linear Programming is a mathematical method used to determine the best possible outcome (like maximum profit or lowest cost) in a given mathematical model whose requirements are represented by linear relationships.

Z

Objective Function

The linear function Z = ax + by which must be maximized or minimized. The variables x and y are called decision variables.

≤ / ≥

Constraints

The linear inequalities or equations that restrict the decision variables based on limited resources (like money, time, or raw materials).

+

Non-Negativity

The specific constraints x ≥ 0 and y ≥ 0. Because physical quantities (like number of chairs produced) cannot be negative, we only work in the First Quadrant.

Part 2

Feasible Regions & Solutions

Bounded Feasible Region
Closed perfectly on all sides by constraints.
Unbounded Feasible Region
Extends infinitely outward. Max/Min may not exist.
  • Feasible Region: The common region determined by all the constraints including the non-negative constraints x, y ≥ 0. Every point in this region satisfies all constraints simultaneously.
  • Feasible Solution: Any point within or on the boundary of the feasible region.
  • Optimal (Feasible) Solution: The specific feasible solution that yields the maximum or minimum value of the objective function Z.
Part 3

Fundamental Theorems of LPP

Theorem 1:
Let R be the feasible region for a linear programming problem and let Z = ax + by be the objective function. When Z has an optimal value (maximum or minimum), it MUST occur at a corner point (vertex) of the feasible region.

Theorem 2:
Let R be the feasible region. If R is bounded, then the objective function Z has both a maximum and a minimum value on R, and each of these occurs at a corner point.
Part 4

The Corner Point Method

This is the standard NCERT algorithm to solve all linear programming problems.

  1. Graph the constraints: Draw the lines for all inequalities and shade the common intersection area in the first quadrant. This is your Feasible Region.
  2. Find the Corner Points: Identify all the vertices of the shaded region. You may need to solve line equations simultaneously to find exact intersections.
  3. Evaluate Z: Plug the (x, y) coordinates of every corner point into the objective function Z = ax + by.
  4. Determine Optimal Value:
    • If the region is Bounded: The highest value is the Maximum, the lowest is the Minimum.
    • If the region is Unbounded: Proceed to the Unbounded Check (below).

⚠️ The Unbounded Region Check

If your feasible region is open/unbounded, the max/min value you found at the corner points might be fake. Let 'M' be the maximum value and 'm' be the minimum value found at the corners.

  • To verify Maximum 'M': Draw the dotted line ax + by > M. If this open half-plane has any points in common with the feasible region, then the LPP has no maximum value. Otherwise, M is the true max.
  • To verify Minimum 'm': Draw the dotted line ax + by < m. If this open half-plane shares any points with the feasible region, then the LPP has no minimum value. Otherwise, m is the true min.
Part 5

Types of Linear Programming Problems

Manufacturing Problems

Determine the number of units of different products to produce in order to maximize profit, given limitations on machine hours, labor, and raw materials.

Diet Problems

Determine the amount of different foods to consume to meet minimum nutritional requirements (vitamins, minerals) while minimizing the total cost of the diet.

Transportation Problems

Find the cheapest way to transport goods from several factories/plants to various distribution centers or markets without exceeding supply or failing demand.

Part 6 (Advanced)

Competitive Method: Iso-Profit Lines

Instead of testing every single corner point, you can visually sweep a line across the graph to find the optimal point instantly. Highly useful for JEE/Competitive time-saving.

Iso-Profit Line Method
The line ax + by = Z is moved parallel to itself. The last vertex it touches before leaving the region is the Maximum.

🎯 Infinite Optimal Solutions

If the Iso-profit/Iso-cost line (the line representing the objective function Z) is exactly parallel to one of the boundary constraint lines of the feasible region, then every single point on that line segment yields the same optimal value.

In this case, the LPP has infinitely many optimal solutions.
