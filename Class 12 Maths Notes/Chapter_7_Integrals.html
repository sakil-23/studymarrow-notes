Chapter 7: Integrals

The heavyweight of Calculus: Anti-derivatives, ILATE, Partial Fractions, Definite Integrals, and the legendary King's Rule.

Part 1

Integration as an Inverse Process

Integration is the exact reverse process of differentiation. Instead of finding the derivative of a function, we are given the derivative and must find the original function (the "anti-derivative").

The Calculus Cycle
Integration restores the function, but adds an arbitrary constant 'C'.
The Arbitrary Constant (C):

Since the derivative of any constant is zero (e.g., d/dx(x² + 5) = 2x, and d/dx(x² - 9) = 2x), integrating 2x gives x² + some unknown constant. We denote this family of curves by adding + C to all indefinite integrals.

Part 3

Methods of Integration

1. Substitution Method

Used when an integral contains a function and its exact derivative.

Let ∫ f(g(x)) · g'(x) dx.
Substitute t = g(x), which means dt = g'(x) dx. The integral becomes ∫ f(t) dt.

2. Integration by Parts

The product rule for integration. Used when two different types of functions are multiplied.

∫ u · v dx = u ∫ v dx - ∫ ( u' · ∫ v dx ) dx

The ILATE Rule
The function that comes first in this word should be set as 'u' (the one to differentiate).

💡 The Golden Exponential Shortcut

A highly tested pattern in board exams and JEE. If you see $e^x$ multiplied by a sum of two functions, check if one is the derivative of the other:

∫ ex [ f(x) + f'(x) ] dx = ex · f(x) + C

3. Partial Fractions

Used to integrate proper rational functions (where the degree of the numerator is strictly less than the denominator).

Rational Form Partial Fraction Decomposition

(px + q) / (x-a)² A/(x-a) + B/(x-a)²
(px² + qx + r) / [ (x-a)(x²+bx+c) ] A/(x-a) + (Bx+C)/(x²+bx+c)
Part 4

Special Integrals (The Cheat Sheet)

Without Square Roots:

• ∫ 1/(x² - a²) dx = (1/2a) · log |(x-a)/(x+a)|
• ∫ 1/(a² - x²) dx = (1/2a) · log |(a+x)/(a-x)|
• ∫ 1/(x² + a²) dx = (1/a) · tan⁻¹(x/a)

With Square Roots in Denominator:

• ∫ 1/√(x² - a²) dx = log |x + √(x² - a²)|
• ∫ 1/√(x² + a²) dx = log |x + √(x² + a²)|
• ∫ 1/√(a² - x²) dx = sin⁻¹(x/a)
Part 5

Definite Integrals & Properties

A definite integral has upper and lower limits. It calculates the exact net area under the curve between those limits. Note that Definite integrals do NOT have a '+ C'.

ab f(x) dx
Fundamental Theorem of Calculus

The Holy Trinity of Properties

1. The King's Rule (Most Important)

∫ab f(x) dx = ∫ab f(a + b - x) dx

Used extensively when the denominator remains the same after applying the rule. You add the original integral 'I' and the new integral 'I' to get 2I.

2. Even & Odd Function Rule

For symmetric limits ∫-aa f(x) dx:

• If f(x) is Odd (f(-x) = -f(x)): Result is exactly 0.
• If f(x) is Even (f(-x) = f(x)): Result is 2 · ∫0a f(x) dx.

3. The Splitting Property (For Modulus)

∫ab f(x) dx = ∫ac f(x) dx + ∫cb f(x) dx

Crucial for modulus functions |x| where the definition of the function breaks/changes at x = c.

Part 6 (Advanced)

Competitive Topics (JEE Level)

🚀 Newton-Leibniz Formula (Differentiation under Integral)

If the limits of a definite integral are functions of x, and you need to differentiate the entire integral with respect to x:

d/dx [ ∫u(x)v(x) f(t) dt ] = f(v(x)) · v'(x) - f(u(x)) · u'(x)

Highly tested in Limits problems involving 0/0 form requiring L'Hôpital's rule!

⚔️ Walli's Formula (Shortcut for Trig Powers)

For integrating strictly from 0 to π/2:

∫0π/2 sinn(x) dx = ∫0π/2 cosn(x) dx

  • If n is Even: [ (n-1)(n-3)...(1) ] / [ n(n-2)...(2) ] × (π/2)
  • If n is Odd: [ (n-1)(n-3)...(2) ] / [ n(n-2)...(3) ] × (1)
