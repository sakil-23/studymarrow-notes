Chapter 1: Relations & Functions

Elevating algebra to logic: Equivalence Relations, Bijective Mappings, Invertibility, and Binary Operations.

Part 1

Types of Relations

Recall from Class 11 that a relation R from set A to itself is a subset of A × A. In Class 12, we categorize these relations based on specific, strict logical behaviors.

1. Reflexive Relation

Every element of A is related to itself.

(a, a) ∈ R for every a ∈ A.

2. Symmetric Relation

If 'a' is related to 'b', then 'b' must be related to 'a'.

(a, b) ∈ R ⇒ (b, a) ∈ R

3. Transitive Relation

If 'a' is related to 'b', and 'b' is related to 'c', then 'a' must be related to 'c'.

(a, b) ∈ R and (b, c) ∈ R ⇒ (a, c) ∈ R

Equivalence Relation:

A relation R on a set A is called an Equivalence Relation if and only if it is simultaneously Reflexive, Symmetric, and Transitive.

Equivalence Class: If R is an equivalence relation on set A, it partitions A into mutually disjoint subsets called equivalence classes. Elements within the same class are related to each other, but not to elements in other classes.

📈 Competitive Shortcuts: Counting Relations

For a set A containing n elements:

  • Total number of relations = 2n²
  • Total Reflexive relations = 2n(n-1)
  • Total Symmetric relations = 2n(n+1)/2
Part 2

Types of Functions (Mappings)

One-One (Injective)
Distinct elements of X map to distinct images in Y.
Onto (Surjective)
Every element in Y has at least one pre-image in X.
  • One-One (Injective): A function f: X → Y is one-one if f(x₁) = f(x₂) ⇒ x₁ = x₂ for all x₁, x₂ ∈ X.
  • Many-One: If two or more elements in X map to the same element in Y.
  • Onto (Surjective): A function is onto if the Range is perfectly equal to the Codomain.
  • Into: If there is at least one element in Y which is NOT an image of any element in X (Range ⊂ Codomain).
  • Bijective: A function that is BOTH One-One AND Onto. This is a critical property for finding inverses.

🎯 The Horizontal Line Test

If you have the graph of a function:

  • If every horizontal line intersects the curve at exactly ONE point, the function is One-One.
  • If any horizontal line intersects the curve at more than one point, it is Many-One.
Part 3

Composition & Invertibility

Composition of Functions ( gof ):

Let f: A → B and g: B → C be two functions. The composition of f and g, denoted by gof, is a function from A → C defined as:

gof(x) = g(f(x)) for all x ∈ A.

*Note: Composition is generally NOT commutative (fog ≠ gof), but it IS associative (ho(gof) = (hog)of).

Invertible Functions

A function f: X → Y is defined to be invertible if there exists a function g: Y → X such that:
gof = IX (Identity function on X)
fog = IY (Identity function on Y)

CRITICAL CONDITION: A function f is invertible if and only if f is a Bijective function (One-One and Onto). If it fails either, it has no inverse!

Part 4

Binary Operations

A binary operation * on a set A is a function from A × A → A. It combines two elements of a set to produce a third element belonging to the same set.

  • Commutative: a * b = b * a for all a, b ∈ A.
  • Associative: (a * b) * c = a * (b * c) for all a, b, c ∈ A.
  • Identity Element (e): An element e is an identity if a * e = a and e * a = a for all a ∈ A. (e.g., 0 is the identity for addition).
  • Invertible Element: An element a is invertible if there exists an element b such that a * b = e and b * a = e. Here, b is the inverse of a.
Part 5 (Advanced)

Function Combinatorics (JEE Level)

If Set A has m elements and Set B has n elements:

Type of Function Total Number Possible Condition
Total Functions nm Always valid
One-One Functions nPm Only if n ≥ m (0 if n < m)
Bijective Functions m! Only if m = n (0 if m ≠ n)
