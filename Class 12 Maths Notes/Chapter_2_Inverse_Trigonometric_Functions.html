Chapter 2: Inverse Trigonometric Functions

Mastering restricted domains, principal value branches, graphical transformations, and advanced conditional identities.

Part 1

The Need for Restricted Domains

From Chapter 1, we know that a function must be Bijective (One-One and Onto) to have an inverse. However, all trigonometric functions are periodic, making them "Many-One". To make them invertible, we must artificially restrict their domains to specific intervals called Principal Value Branches.

Function Domain (Input) Principal Value Branch (Range/Output)
y = sin⁻¹(x) [-1, 1] [-π/2, π/2]
y = cos⁻¹(x) [-1, 1] [0, π]
y = tan⁻¹(x) R (All Real Numbers) (-π/2, π/2)
y = cot⁻¹(x) R (All Real Numbers) (0, π)
y = sec⁻¹(x) R - (-1, 1) [0, π] - {π/2}
y = cosec⁻¹(x) R - (-1, 1) [-π/2, π/2] - {0}

Part 3

Core Properties & Negative Angles

1. Self-Adjusting Property

• sin(sin⁻¹ x) = x   [For x ∈ [-1, 1]]
• sin⁻¹(sin θ) = θ   [ONLY if θ ∈ [-π/2, π/2]]

*If θ is outside the principal branch, you must use allied angles to bring it inside before canceling!

2. Reciprocal Property

• cosec⁻¹(x) = sin⁻¹(1/x)   [|x| ≥ 1]
• sec⁻¹(x) = cos⁻¹(1/x)   [|x| ≥ 1]
• cot⁻¹(x) = tan⁻¹(1/x)   [If x > 0]
  If x < 0, cot⁻¹(x) = π + tan⁻¹(1/x)

3. Negative Angle Properties (Odd/Even Analogs):

"The Sine Family" (Pulls negative out directly):
• sin⁻¹(-x) = -sin⁻¹(x)
• tan⁻¹(-x) = -tan⁻¹(x)
• cosec⁻¹(-x) = -cosec⁻¹(x)

"The Cosine Family" (Subtracts from π):
• cos⁻¹(-x) = π - cos⁻¹(x)
• cot⁻¹(-x) = π - cot⁻¹(x)
• sec⁻¹(-x) = π - sec⁻¹(x)
Part 4

Operational Identities & Competitive Formulas

4. Complementary Angles

• sin⁻¹(x) + cos⁻¹(x) = π / 2    [x ∈ [-1, 1]]
• tan⁻¹(x) + cot⁻¹(x) = π / 2    [x ∈ R]
• sec⁻¹(x) + cosec⁻¹(x) = π / 2    [|x| ≥ 1]

🔥 5. Sum & Difference Formulas (JEE Core)

For tan⁻¹(x) + tan⁻¹(y):

  • If xy < 1 (NCERT Standard):
    tan⁻¹(x) + tan⁻¹(y) = tan⁻¹[ (x + y) / (1 - xy) ]
  • If xy > 1 (and x>0, y>0) (Advanced):
    tan⁻¹(x) + tan⁻¹(y) = π + tan⁻¹[ (x + y) / (1 - xy) ]

For Subtraction:
tan⁻¹(x) - tan⁻¹(y) = tan⁻¹[ (x - y) / (1 + xy) ]    (Valid for xy > -1)

6. Multiple Angle Conversions:

The term 2 tan⁻¹(x) is highly versatile and can be converted into any other inverse function (assuming appropriate domains like |x| ≤ 1):

• 2 tan⁻¹(x) = sin⁻¹ [ 2x / (1 + x²) ]
• 2 tan⁻¹(x) = cos⁻¹ [ (1 - x²) / (1 + x²) ]
• 2 tan⁻¹(x) = tan⁻¹ [ 2x / (1 - x²) ]
Part 5

Optimal Substitution Strategies

To simplify complex inverse trigonometric expressions (especially in calculus), we use standard trigonometric substitutions.

If you see this Algebraic Expression... Substitute this Trigonometric Ratio
√(a² - x²) x = a sin(θ)   or   x = a cos(θ)
√(a² + x²) x = a tan(θ)   or   x = a cot(θ)
√(x² - a²) x = a sec(θ)   or   x = a cosec(θ)
√(a - x) / (a + x)   or   √(a + x) / (a - x) x = a cos(2θ)
√(a² - x²) / (a² + x²) x² = a² cos(2θ)
